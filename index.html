Projects

LangRec

Speaking multiple languages is a valuable skill, but it can be costly and challenging to find the best resources to study a foreign language. LangRec provides users with targeted recommendations for language-learning resources based on their learning style, language goals, budget and current level. LangRec is a server side application built using React and Node. Some additional skills used include:

  • React's Context API
  • Express
  • PostgreSQL

MyHealthHacker

MyHealthHacker helps users to keep track of their health-related habits such as sleep, exercise, and hydration, as well as their mental health. Over time, they can track trends in their physical and mental health and adjust their behavior to feel their best! The data is managed using a PostgreSQL database. Additional skills used include:

  • React
  • Node
  • Express
Github Client Repo Github API Repo Visit MyHealthHacker

Story Seeker

Most people have experienced the feeling of finishing a great book and not knowing what to read next. Story Seeker helps users to find the perfect book that they will love to read! Using Story Seeker, users can submit the title of a book that they enjoy. Story Seeker then provides book recommendations that the user will likely enjoy and shows purchase options. I utilized the TasteDive API and the Google Books API to provide users with recommendations. Additional skills used include:

  • JavaScript ES6
  • jQuery
  • HTML/CSS
